Longitudinal Climate Change Analysis

There have been recent arguements made against the existance of climate change and its impacts. This report analyses the Mauna Lao Observatory data set recording atmospheric carbon dioxide concentrations since the year 1960 to present day and Sable Island data observing temperature data

The analysis was written in R using tools: tidyverse, ggplot2, and INLA. You can find the full analysis here with the full code here
